# Data

Datos anonimizados de consensos, publicados como parte de la transparencia radical del proyecto (ver [docs/01-problema-y-vision.md](../docs/01-problema-y-vision.md)).

Se poblará a partir de la Etapa 4 de la hoja de ruta (ciclos de votación + registro histórico), con un dataset por ciclo. Ningún dato personal identificable se publica aquí: solo síntesis agregadas y resultados de votación.
